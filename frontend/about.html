Sobre o Projeto

O CineViewExperience é um site de cinema fictício criado para fins educacionais, com foco em segurança ofensiva.

A ideia foi simular uma plataforma real, com cadastro, login, compra de ingressos e tudo mais que um sistema assim teria — só que com falhas propositalmente inseridas pra quem quiser aprender segurança na prática.

O site foi feito com Python puro no backend e HTML, CSS e JavaScript no frontend, usando banco de dados PostgreSQL e rodando via Docker, pra facilitar na hora de subir tudo local.

É um ambiente pensado pra testes, então tem vulnerabilidades comuns de propósito. Serve tanto pra quem tá começando quanto pra quem quer treinar com um sistema mais completo.
